# 🚀 온디바이스(On-Device) DSM 시스템
** 개발자: j2w **

이 프로젝트는 1GB RAM의 라즈베리파이 3 환경에서 구동되는 초저전력 임베디드 운전자 상태 감시 시스템(DSM)입니다.

## 🌟 핵심 기술 (Features)
- 🧠 **결정 수준 융합 (Decision-level Fusion):** 딥러닝(81.3%)과 기하학적 수학 로직(18.7%)의 완벽한 융합 방어!
- ⚡ **Zero-Latency:** 무거운 통신망을 버리고 I2C LCD 패널로 0.0ms 즉각 출력!
- 🗜️ **초경량화:** INT8 양자화(Quantization) 및 NCNN 모바일 엔진 적용!

## ⚙️ 하드웨어 요구사항 (Hardware Requirements)
- Raspberry Pi 3 Model B (1GB RAM)
- Raspberry Pi Camera Rev 1.3
- 16x2 LCD Panel (I2C)

## 🛠️ 실행 방법 (How to Run)
1. 라즈베리파이3에 전원만 연결하면 실행이 됩니다.
2. 종료를 하고 싶으면 전원 연결을 해제하거나 카메라를 보고 일정한 행동을 취하는 이스터에그를 찾아서 실행하면 됩니다.
